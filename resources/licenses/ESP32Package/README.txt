## ESP32 Arduino Package

Dieses Repository enthält das ESP32-Arduino-Board-Package
**in genau der Form, wie es durch den Arduino Boardmanager installiert wurde**.
Das Paket wurde **nicht verändert**, sondern lediglich als Bestandteil dieser
Entwicklungsumgebung übernommen, um die Nutzung der IoT-Werkstatt-Beispiele
zu vereinfachen.

**Quelle (Upstream-Projekt):**  
https://github.com/espressif/arduino-esp32

### Lizenzhinweis

Das ESP32-Arduino-Package besteht aus verschiedenen Komponenten
(ESP32 Arduino Core, Toolchain, esptool, SDK-Bestandteile, Bibliotheken usw.),
die jeweils unter ihren **ursprünglichen Open-Source-Lizenzen** stehen.

Da das vollständige Paket unverändert aus der Boardmanager-Installation
übernommen wurde, kann dieses Repository **keine Aussagen darüber machen,
welche Lizenzdateien im Installationspaket enthalten oder nicht enthalten sind**.
Für die Lizenzierung gelten ausschließlich die Bedingungen der jeweiligen
Originalautoren.

Bitte beachten Sie die Lizenzhinweise im Upstream-Repository:
https://github.com/espressif/arduino-esp32
