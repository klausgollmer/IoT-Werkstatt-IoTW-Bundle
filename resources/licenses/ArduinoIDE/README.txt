## Arduino IDE 1.8.19

Dieses Repository enthält eine Kopie der Arduino IDE 1.8.19
**in genau der Form, wie sie von der offiziellen Arduino-Website bereitgestellt wird**.
Die IDE wurde **nicht verändert** und ausschließlich zur vereinfachten Nutzung
im Rahmen der IoT-Werkstatt-Laufzeitumgebung eingebunden.

**Quelle (Originalprojekt):**  
https://www.arduino.cc/en/software

### Lizenzhinweis

Die Arduino IDE 1.x ist ein Open-Source-Projekt und besteht aus verschiedenen
Komponenten (Java-Anwendungen, Arduino-Core-Bibliotheken, Compiler-Toolchains,
Hilfsprogramme usw.), die jeweils unter ihren **ursprünglichen Lizenzen** stehen.

Da diese IDE-Version unverändert aus dem offiziellen Download übernommen wurde,
kann dieses Repository **keine Aussagen darüber treffen**, welche einzelnen
Lizenzdateien im gelieferten Paket enthalten oder nicht enthalten sind.
Die vollständigen Lizenzinformationen entnehmen Sie bitte den Angaben des
Originalprojekts.

Es gelten ausschließlich die Lizenzbedingungen und Urheberrechte der Arduino-Projektentwickler und der jeweiligen Drittanbieterkomponenten.
