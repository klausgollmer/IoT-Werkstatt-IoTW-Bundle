## Arduino Libraries (aus dem Arduino Bibliotheksmanager)

Dieses Repository enthält eine Reihe von Arduino-Bibliotheken,
**in genau der Form, wie sie vom Arduino Bibliotheksmanager installiert wurden**.
Die Bibliotheken wurden **nicht verändert** und ausschließlich zur
Vereinfachung der Nutzung der IoT-Werkstatt-Beispiele in diese
Laufzeitumgebung aufgenommen.

### Herkunft
Die Bibliotheken stammen direkt aus dem offiziellen Arduino
Bibliotheksmanager sowie den darin enthaltenen externen Quellen
(GitHub-Repositories der jeweiligen Autoren).

### Lizenzhinweis

Jede dieser Bibliotheken unterliegt den **ursprünglichen Lizenzen der jeweiligen Autoren**
(z. B. MIT, BSD, GPL, LGPL, Apache oder proprietäre Lizenzen).  
Da das Repository die Bibliotheken **unverändert** aus der lokalen Arduino-
Installation übernimmt, kann **keine Aussage darüber getroffen werden**, welche
Lizenzdateien die jeweiligen Bibliotheksarchive enthalten oder nicht enthalten.

Für alle Lizenzinformationen gelten ausschließlich die Lizenztexte und
Hinweise, die von den Originalautoren in ihren jeweiligen Repositories
bereitgestellt werden.

Die Quellen der eingesetzten Bibliotheken finden Sie im Arduino
Bibliotheksmanager oder direkt in den verlinkten Projekten der jeweiligen
Autoren.
